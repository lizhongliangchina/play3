# 双人情侣默契度测试游戏 - 部署包

## 项目说明
这是一个优化后的部署包，包含双人情侣默契度测试游戏的所有必要文件。

## 部署方式

### 方式一：Vercel 手动部署
1. 访问 [vercel.com](https://vercel.com)
2. 登录或注册账号
3. 点击 "New Project"
4. 将此文件夹拖拽到部署区域
5. 点击 "Deploy"

### 方式二：Vercel CLI 部署
```bash
npx vercel --prod
```

### 方式三：Netlify 部署
1. 访问 [netlify.com](https://netlify.com)
2. 登录账号
3. 将此文件夹拖拽到部署区域
